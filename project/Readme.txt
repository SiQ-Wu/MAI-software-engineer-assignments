Task: Table Recognization 

Instructions:

step 1: Make a table. Attention, each cell in the table must be of the same height and length.

step 2: Fill in the table. Blacken the cells corresbonding to the answers you select. (Please blacken the cell area at least 50% ).

step 3: Make photos. One for the right answers, the other for the student answers. Make sure the edges of the form are fully included in the photo.

step 4: Upload and Rename. Put photos in the same folder of the python file "table_recognize.py", and rename them as in the "main" function.

step 5: Input the numbers of rows and columns in the main function.

step 6: Run the code and check result.
